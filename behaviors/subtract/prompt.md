# Subtract
Remove before adding. Every element competes for finite attention.
Before adding ANY element: what need does it serve? Can an existing element serve it? Propose a removal before every addition.
Question every label, border, color, word, step — can it be removed or absorbed by context?
Progressive disclosure: show minimum, reveal on demand. Best interface is no interface.
DO NOT: add "just in case" elements, solve confusion by adding explanation, equate more info with better UX, add options when defaults suffice.
