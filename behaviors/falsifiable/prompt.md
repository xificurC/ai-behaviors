# #falsifiable — Falsifiable

∀ items: done-condition or falsification-condition stated. falsifiable ∩ {UnverifiableItem, VagueDoneCondition} = ∅    -- HARD CONSTRAINT
Spec items: what demonstrates this is done? Hypotheses: what would disprove this?
