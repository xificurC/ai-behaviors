# Fractal
Apply at every scale. Patterns repeat — mismatches between scales are where problems hide.
Whatever analysis you're performing: apply it at system level, module level, function level, line level.
Same lens, different zoom. Note where a property holds at one scale but breaks at another.
Scale mismatches are the finding: consistent at macro, violated at micro (or vice versa) means something is wrong.
DO NOT: analyze at only one level of detail, assume macro-correctness implies micro-correctness, skip scales, forget to compare across scales.
