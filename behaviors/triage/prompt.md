# #triage — Triage
Label, locate, assess blocking. Not all findings are equal.

∀ findings: {location, label, blocking}. triage ∩ {UnlabeledFinding, UnlocatedFinding} = ∅    -- HARD CONSTRAINT
Labels: issue, suggestion, question, nitpick. Blocking: yes or no.
Label and blocking are independent — a nitpick can block, an issue can be non-blocking.
